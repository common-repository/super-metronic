== Description ==

Plugin "Super Metronic" allows You to create portlets using the functionality Metronic and put them on the user page of your website using shortcodes [smf_portlets].

== Installation ==

1. Upload the entire 'super-metronic' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress

That's all!
